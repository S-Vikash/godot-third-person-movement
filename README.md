# Godot Third Person Movement

A simple third-person character controller built in Godot.

## Features

- Camera-relative movement
- Character jumping
- Smooth player controls
- Third-person camera using spring arm
- Reload level on fall

https://github.com/user-attachments/assets/9dba08e0-ef20-4216-b6c9-ba40a2cb1181


## Controls

| Key | Action |
|------|--------|
| W A S D | Move |
| Mouse | Look Around |
| Space | Jump |

## Engine

Godot 4.x

## How to Run

1. Clone the repository.
2. Open the project in Godot.
3. Run the main scene.
